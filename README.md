A YARD [plugin](https://rubydoc.info/gems/yard/file/docs/GettingStarted.md#Plugin_Support) that parses Sorbet type annotations

## Install

```shell
gem install yard-sorbet
```

## Usage
```bash
yard doc --plugin sorbet
```
